# 七政四余技能

> "七政四余，星曜推命。" —— 果老星宗

七政四余是中国传统星命术数，以七政（日、月、金、木、水、火、土）和四余（罗睺、计都、月孛、紫气）星曜落宫，进行命运推算。本技能提供完整的七政四余星盘排盘和推命功能。

## ✨ 核心功能

- ✨ 七政星曜排布（太阳、太阴、金星、木星、水星、火星、土星）
- ✨ 四余星曜排布（罗睺、计都、月孛、紫气）
- ✨ 十二宫位定位（命宫、财帛、兄弟等）
- ✨ 星曜亮度计算（庙/旺/得地/陷/弱）
- ✨ 星曜关系分析（相生/相克/相合）
- ✨ 果老星宗推命（性格/事业/婚姻/财运）
- ✨ 纯 Python 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确星曜位置计算（1900-2100 年）
- 🔧 完整十二宫位算法
- 🔧 星曜亮度判断
- 🔧 纯 Python 实现，无外部依赖
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/qizheng-siyu-skill.git
cd qizheng-siyu-skill

# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +x scripts/qizheng_pan.py
```

### 基本使用

```bash
# 基础排盘（按出生时间）
python3 scripts/qizheng_pan.py --date "1990-05-15" --hour 8

# 指定性别
python3 scripts/qizheng_pan.py --date "1990-05-15" --hour 8 --gender 女

# 指定问事类型（带推命分析）
python3 scripts/qizheng_pan.py --date "1990-05-15" --hour 8 --question 财运

# 输出 JSON 格式
python3 scripts/qizheng_pan.py --date "1990-05-15" --hour 8 --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 出生日期 (YYYY-MM-DD) | 必填 |

| --gender | -g | 性别 (男/女) | 男 |
| --question | -q | 问事类型 | 通用 |

## 📚 七政四余基础知识

### 七政星曜

| 星曜 | 别名 | 五行 | 特性 |
|------|------|------|------|
| 太阳 | 日 | 火 | 光明、权威、父亲 |
| 太阴 | 月 | 水 | 柔和、情感、母亲 |
| 金星 | 太白 | 金 | 温润、艺术、财富 |
| 木星 | 岁星 | 木 | 仁慈、扩张、贵人 |
| 水星 | 辰星 | 水 | 智慧、沟通、贸易 |
| 火星 | 荧惑 | 火 | 热情、冲动、行动 |
| 土星 | 镇星 | 土 | 稳重、压力、责任 |

### 四余星曜

| 星曜 | 别名 | 五行 | 特性 |
|------|------|------|------|
| 罗睺 | 北交点 | 火 | 变化、突发、意外 |
| 计都 | 南交点 | 土 | 稳定、累积、阻碍 |
| 月孛 | 月球远地点 | 水 | 情感、波动、桃花 |
| 紫气 | 月球近地点 | 木 | 贵气、气质、神秘 |

### 星曜亮度

| 亮度 | 含义 | 吉凶 |
|------|------|------|
| 庙 | 入庙，最强 | 大吉 |
| 旺 | 旺相，强 | 吉 |
| 得地 | 得地，中等 | 小吉 |
| 陷 | 落陷，弱 | 凶 |
| 弱 | 无力，最弱 | 大凶 |

### 十二宫位

| 宫位 | 含义 |
|------|------|
| 命宫 | 性格、命运、外貌 |
| 财帛 | 财运、收入、理财 |
| 兄弟 | 兄弟姐妹、朋友 |
| 田宅 | 房产、家庭、不动产 |
| 男女 | 子女、晚辈 |
| 奴仆 | 下属、员工、服务 |
| 夫妻 | 婚姻、配偶、感情 |
| 疾厄 | 健康、疾病 |
| 迁移 | 外出、迁移、社交 |
| 官禄 | 事业、工作、地位 |
| 福德 | 福气、精神享受 |
| 父母 | 父母、长辈、遗传 |

### 星曜关系

| 关系 | 五行 | 含义 |
|------|------|------|
| 相生 | 木→火→土→金→水→木 | 增益、帮助 |
| 相克 | 木→土→水→火→金→木 | 克制、压制 |
| 相合 | 同五行或六合 | 和谐、合作 |
